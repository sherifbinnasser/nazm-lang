## تعبيرات الفهارس

تُستخدم تعبيرات الفهارس في لغة **نظم** للوصول إلى عنصر معين داخل المصفوفة أو أي بنية أخرى قابلة للفهرسة باستخدام الفهرس. يُسهّل هذا النوع من التعبيرات معالجة البيانات بطريقة منظمة وفعّالة، حيث يُمكن من خلاله الوصول إلى العناصر مباشرة عن طريق موضعها، إليك مثال لفهرسة المصفوفات:

```nazm
دالة البداية() {

    احجز أعداد = [30، 80، 50، 200، 8000]؛

    احجز الثاني = أعداد[1]؛
    اظهر_(الثاني)؛ // سيعرض 80

    احجز فهرس = 4ط؛
    احجز الأخير = أعداد[فهرس]؛
    اظهر_(الأخير)؛ // سيعرض 8000

    // خطأ أثناء التنفيذ: الفهرس 6 أكبر من حجم المصفوفة 5
    احجز فهرس_غير_صالح = أعداد[6]؛

}
```

يمكن أيضاً أن تستخدم تعبيرات الفهارس مع السلاسل والمؤشرات لقراءة محتواها بعد إزاحة معينة:


```nazm
دالة البداية() {

    احجز كلام: [ط1] = "9875"؛

    احجز الثاني: ط1 = كلام[1]؛
    اظهر_(الثاني)؛ // سيعرض 8

    احجز م = كلام.مؤشر؛
    احجز الأخير = م[3]؛
    اظهر_(الأخير)؛ // سيعرض 5

    // قد يظهر خطأ الوصول للذاكرة إذا كان المؤشر لا يمكن الوصول له من برنامجك
    احجز قيمة_ما = أعداد[6]؛

}
```
